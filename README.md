# QuickDAO4

> 一个强大，简单，易用的Java ORM框架

- 一个DAO对象完成所有数据库操作
- 支持数据库自动建表，自动新增字段
- 支持外键关联查询
- 支持无实体类查询
- 根据数据库表逆向生成Java实体类
